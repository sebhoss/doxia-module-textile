h1. Doxia Textile Module "!https://secure.travis-ci.org/sebhoss/doxia-module-textile.png!":http://travis-ci.org/sebhoss/doxia-module-textile "!https://www.ohloh.net/p/doxia-module-textile/widgets/project_thin_badge.gif!":https://www.ohloh.net/p/doxia-module-textile/

p. This "Doxia":http://maven.apache.org/doxia/ module enables you to write documentation using the "Textile":http://txstyle.org/ syntax. It is based on the "Mylyn/WikiText":http://wiki.eclipse.org/Mylyn/WikiText project which also provides a nice editor with syntax completion, folding, validation, outline view and a preview of what the HTML output looks like.

h2. Usage

p. Simply reference this module as a dependency to the maven-site-plugin inside your POM:

bc. 
<plugin>
  <groupId>org.apache.maven.plugins</groupId>
  <artifactId>maven-site-plugin</artifactId>
  <dependencies>
    <dependency>
      <groupId>com.github.sebhoss.doxia</groupId>
      <artifactId>doxia-module-textile</artifactId>
      <version>1.0.0</version>
    </dependency>
  </dependencies>
</plugin>

p. All Textile source documents with the file extension *textile* living under *src/site/textile/* will be parsed and converted to HTML.
